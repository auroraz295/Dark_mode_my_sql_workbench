## Dark Mode per MySQL Workbench
Questa guida mostra come installare una dark mode personalizzata per MySQL Workbench sostituendo il file del code editor.

### Installazione Dark Mode
<h4> 1. Scaricare il file della dark mode </h4>
Scaricare il file:
code_editor_dark_mode.xml

<h4> 2. Aprire la cartella di installazione di MySQL Workbench </h4>
Aprire Esplora File e seguire questo percorso:
Questo PC > Disco Locale (C:) > Programmi > MySQL > MySQL Workbench 8.0 CE > data
<br> In alcune versioni, la cartella può chiamarsi:
MySQL Workbench CE invece di MySQL Workbench 8.0 CE.

<h4> 3. Rinominare il file scaricato </h4>
Rinominare il file:
code_editor_dark_mode.xml
in:
code_editor.xml

<h4> 4. Sostituire il file originale </h4>
Copiare il file rinominato all’interno della cartella:
data
Windows chiederà se si desidera sostituire il file esistente: confermare.

<h4> 5. Avviare MySQL Workbench </h4>
Aprire MySQL Workbench normalmente.
La dark mode sarà ora applicata al code editor.

## Ripristinare il Tema Originale
Se si desidera tornare al tema originale di MySQL Workbench:

<h4> 1. Scaricare il file originale </h4>
Scaricare:
code_editor_originale.xml

<h4> 2. Rinominare il file </h4>
Rinominare il file in:
code_editor.xml

<h4> 3. Tornare nella cartella data </h4>
Aprire il percorso:
Questo PC > Disco Locale (C:) > Programmi > MySQL > MySQL Workbench 8.0 CE > data

<h4> 4. Sostituire il file </h4>
Copiare il file rinominato nella cartella data e sostituire quello esistente.

<h4> 5. Riavviare MySQL Workbench </h4>
Aprire nuovamente MySQL Workbench per tornare al tema originale.
